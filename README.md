# 2021年度之「最」分享：书籍、影视剧、产品、金句

本分享来自于范冰在他的 [newsletter](https://us6.campaign-archive.com/home/?u=e4582460499f4aadae1a90e2b&id=d2c14f8a94) 中的问卷调查，这里我收集整理出来。

包含：

- 推荐一本  2021 读过最好的「书」
- 推荐一部  2021 看过最好的「电影/剧集/动画/综艺」
- 推荐一件  2021 购买的、让你最满意的「产品」
- 回想一句  2021 给你印象最深刻的「金句」
- 自问自答一个你想分享的  2021 年度之「最」

| 问题1：请推荐一本  2021 读过最好的「书」                     |
| ------------------------------------------------------------ |
| 《The Almanack  of Naval Ravikant》  这本书还是看到冰哥即刻上给另一个小伙伴推荐的，于是下载下来看看。里面有太多价值百万的思想，对今年的自己影响最深的有两个 "Productize  yourself. ← 重新审视了目前的工作内容以及自己对于工作的态度" "Read what you love until you  love to read & Read from the original. ← 指导了自己从一个完全不爱看书到去年大概读了接近50本书的过程  & 重新整理自己的信息源" |
| 推荐《微积分的力量》。大学的时候高数学的一塌糊涂，这本科普书第一次让我感觉数学是那么的有趣和好玩，还有历史上的那些鼎鼎大名的数学家，每一个都性格鲜明，在前人的基础之上一步步去建造了一座大厦。而且发现生活中处处可以看到数学的身影，数学不再是枯燥难懂的符号公式，而是解构分析一切事物的工具和方法，真正的科学之根。 |
| 《卡片笔记写作法》      这本书今年非常火，书本身的内容优秀但不是我推荐它作为2021读过「最好的书」的原因。推荐这本书的根本原因在于它改变了我的想法，让我去思考了很多以前没有想过的事情。      在读这本书之前，从未想过曾经读过的书、在书上圈圈画画写下的笔记、亦或是认真摘录下来的一些文字到底最后于我产生了怎样的影响；在读这本书之前也未曾想过，在这个无处不充满信息的时代，在这个时刻都在产生新想法的时代，信息的流动到底于我留下了什么价值。     这本书或许并不会直接告诉我这些问题的答案，但是它对于2021年的我是一个冲击，让我对过去的一些想法产生了疑问，并开始寻找答案。因此我推荐这本书作为2021年我认为最好的书。 |

| 《置身事内》：中国政府与经济发展     原因：在中国生活、工作、投资怎能不了解政府的运作模式，也是闲谈装逼利器。本书有诚意，说人话，阅读体验不错。 |
| 《黑箱》，书的内容我就不做文字介绍了。为什么选这本书，我觉得有时候好还是不好需要结合当前时代和环境作为背景。每次读完这种一人之力对抗上层权利的内容，我都深深感慨，我们需要一个公正的、敢于讲述客观事实的新闻媒体，我们还需要一个不被权利控制的社交媒体，而这两样都不存在。当某个力量控制了人们的所听所看，那些受害者很容易被黑暗和绝望打倒。 |
| 《反脆弱：从不确定性中获益》,  作者名称：纳西姆·尼古拉斯·塔勒布      推荐原因：本书提出"反脆弱性”的概念，角度新颖，让我开始重新思考“稳定”的定义。稳定程度只能代表抗风险的能力，而反脆弱程度能代表遭受风险后恢复的能力，前者人人皆知，  后者往往被忽略，意识到这一点是读本书最大的收获。     书中多讨论了多个有趣观点，少许举例如下：     1.多数人直觉上认为的”稳定"可能恰是脆弱的，比如一些小银行，或者近期出问题的恒大，看起来大而不倒，可一旦被击中阿喀琉斯之踵，  就再也站不起来，反而一些小公司，尤其互联网领域，快速起步，快速失败，快速重来，在动荡中反而不停迭代，难以击倒，反脆弱性更强。      2.局部的破坏可能带来整体系统的提升。用森林火灾比喻：突发的小火灾可以清除森林中易燃的部分，反而减少森林发大火燃烧殆尽的概率，如果森林管控百分百严格，没有任何小火灾，易燃隐患积累，反而更容易产生大火灾。这个例子跟健身的理论恰巧相通：人用阻抗训练摧毁了局部肌肉组织，身体便产生更强力的肌肉组织来应对。客观来讲，某些医疗研究领域，尤其新药新产品等，也适用这个道理。时常轻度”破坏“自己，是个好习惯。     3.潜意识里，人更愿意相信有逻辑的事件，而非随机的事件。为了迎合这个本能，人甚至会强行对事物进行归因，即便毫无根据。                另外：直播里好像听冰哥提到过这本书，所以冰哥可能已经看过了。这个推荐就没有任何价值了，所以我顺手再推荐一本：瑞·达利欧写的《原则》，算理论书也算工具书，我觉得对自我认知有帮助。 |
| Tim  Ferriss的Tribe of  Mentors（台灣譯：人生給的答案），在2021年遇到不如意的時刻，隨意翻一頁都可以感受到滿滿的力量，內心也會感到很平靜，讓自己找回重新回到生活的勇氣，這是一本充滿智慧的大匯總，隨時放在書架上重讀也非常有意思。 |
| 《别独自用餐》      理由：两极分化的评判，只谈适合现阶段的我。这本书颠覆了我对社交的过往看法，我认为社交无用，个人努力才是关键。但现实是即便你的个人能力很强，你依然需要同事、下属、领导的帮助才能上升，同时需要情感的牵连与慰藉。这本书谈的是从思想改变自己，让自己能够提供价值，并且运用社交技巧，利用数字时代的联系，将与他人的交情由浅到深。 |
| 《一个投资家的二十年》通过写专栏阐述投资的过程的一些心得体会，无形中透露着股票投资必须遵守自己先前设定的交易规则和买卖假设，彰显了投资只不过是过程，最终是让自己的生活有得选择。 |
| <好战略坏战略>,个人的经历与生活可以用按部就班来概括,  固执的性格，大多数情况下不撞南墙不回头的惯性决策思路。很少客观, 中立的去思考事情的前后与两面性。从工程师进入商业环境，  原来的线性决策习惯则无法应对如此复杂多变的商业环境，决策则要非常的考究, 恰好这本书提供了一个基础的框架。 |
| 《全球电商进化史》前阿里巴巴副总裁写的，本书整体介绍了电商在各国家的发展进程及潜力。帮我从另外的角度了解了电商行业和阿里。 |
| 在2021年度最有感的閱讀，卡片筆記寫作法：如何實現從閱讀到寫作，這本書直接打破我在做電子筆記的盲點，利用雙向連結的方式，可以串連不同的事情與資料，結合出來完整的邏輯圖。而這套系統在業務上面的處理，也十分好用。再操作B2B業務上面，寫下客戶的筆記是很正常的事情，但是筆記寫下來沒有連結，資料就是很散的存在，可能在A客戶提到B產品，又在C客戶提到B產品，利用卡盒筆記術的原理，利用B產品與A客戶跟C客戶連結起來，將零散的筆記串連成網狀的筆記結構，將筆記整理起來，可以提供的訊息很多。 |
| 《你想活出怎样的人生》，吉野源三郎      这可能是一本被书名耽误的好书（对我来说是如此），初看是浓浓的鸡汤位，看到发现的确是有鸡汤，但同时还有非常有嚼劲的“骨肉”。比如下面这段话——           “无论何时都从你真实的感受、真正打动你的事物出发，不断思考其中的意义。……认真思考在什么样的场合下对哪些事情有感受、感受如何，这样你就能通过具体的感受，或者说无法重复的、仅此一次的经历得到意义，而这种感受不仅限于当时。这就是你真正的思想。……如果你有一丝糊弄，那么无论你曾想到或说出多么了不起的事，都不是真的。” |
| 《我们赖以生存的隐喻》：介绍语言中的“比喻”对思维的影响，依赖比喻固然容易理解，但也不自知地受到暗示。这本书也提醒读者对思维模式本身保持觉察，细想非常有启发。 |
| 《当代中国阶级阶层的变动（1978—2018）》：2021的经济似乎比20的时候大家想象的还要再悲观一些，但也不影响消费升级和消费降级的同时增长，社会矛盾看起来更加激化。这本写于疫情前一年来自社科院的书，可以作为一个观察社会变化的锚点。 |
| 《绝叫》。跟《白夜行》同为所谓社会派推理类型的杰作。相比《白夜行》，《绝叫》有完整的人物成长变化轨迹，行为逻辑真实可信，整个故事更为现实，很有代入感，更为震撼。 |
| 刘子超的《失落的卫星》      跟着作者的脚步，在中亚几个国家中穿梭，体验到的不仅仅是异域风情，更令人印象深刻的是生活在那片土地上，被国家、宗教、民族、身份、历史这些概念所缠绕的人们。相比历史、国家、政治之类的宏大叙事，书中对每一个个体的描写，会让人产生了很多遐想，撩拨着读者，让我们也产生去中亚旅行的冲动。 |
| 《一个农民的亿万传奇》傅海棠口述     普通农民成为期票大鳄的过程，朴实无华，讲的就是普通道理和基本规律。 |

| 《清华园日记》，推荐原因：这本书其实是一本随时都能翻开读两页的书，在大师还没有成长为大师的少年时代，他的大学四年是怎么度过的？是不是也是像我们现在的大学生一样：上不完的课、考不完的试、睡不完的觉、聊不完的天？读完这本八九十年前在清华园上大学的季羡林先生的鲜活真实的日记，你会发现是的，但也不是！说是的是因为大师也曾年轻过，也曾是普通人，也和我们一样有平淡空虚懒散困惑的时候，他会在日记里骂脏话，吐槽老师，吐槽考试，“没做什么有意义的事——妈的，这些混蛋教授，不但不知道自己泄气，还整天考，不是你考，就是我考，考他娘的什么东西？”；坦言自己看球赛是去看大腿，“过午看女子篮球赛，不是去看想打篮球，我想，只是去看大腿。因为说到篮球，实在打得不好”；天气好的时候不想念书只想睡觉，“过午出去走了走，觉得天气太好了。结论是这样的天气还能念书吗？于是回来大睡其觉”。好多好多吐槽调侃的句子现在的大学生读了也会忍俊不禁会心一笑，原来学生时代大家的烦恼基本都差不多。          但是说不是是因为除了那些让人觉得“反差萌”的细节，在每天的日记里，你又能看到他是如何孜孜不倦地读书，他几乎每天都在看书，德文法文英文的原版名著，从未停止过阅读和写作。有一次在精神萎靡的时候，朋友长之和他谈起王国维的刻苦励学，说：“一个大学者的成就并不怎样神奇，其实平淡的很，只是一步步走上去的”，因着这句话，他回家后心情大变，日记里振奋地写道：”I  have gotten  refreshment（我恢复了精神）。”其实，看完季羡林先生这本真实无比的求学日记，才更加用他自己的经历向我们印证了什么叫“其实平淡的很，只是一步步走上去的。” |
| SaaS创业路线图          对于初入 SaaS 行业的技术同学, 这本书比较全面, 也接地气 |
| 书籍：《决策思维：人人必备的决策口袋书》     作者：王嘉陵          推荐理由：      因为工作原因，最近一年每周都需要快速做大量决策。为了提高决策的效率，我阅读了大量关于决策的文章、书籍，而这本书是我感觉和我契合度最高的。当我实践了书里的决策框架，发现决策的效率得到了明显的提升。          书籍不厚，大约一下午就可以读完。如果你正在构建自己的决策框架，相信这本书可以带给你一些启发。 |
| 《加缪手记（第一卷）》     这本书的内容来自加缪的日常记录，一共三卷。第一卷内有加缪写的一封回信《给绝望者的一封信》，第一次读这封信的时候，不夸张的说，我有点冒冷汗。      这封信是这样的，有个参加二战的人给加缪写信，说自己绝望了，加缪回信说，你不应该绝望，你还有很多事情可以做，希特勒鼓动洗脑民众，你也可以在自己的圈子里去影响别人，去跟希特勒对抗，你可以影响十个人、二十个人、三十个人，让大家认同和平才是应该守护的，而不是通过流血解决问题。虽然你感觉一个人的力量很渺小，但是你影响这十个人，这十个人再去影响他们的圈子，这件事是可以做到的。只有当你所有能做的事情都做了，那个时候你可以找个地方，想怎么绝望就怎么绝望，可是现在还远远没到绝望的时候。     加缪的理想主义，完完全全吓到我了，感觉加缪在信里说的是一件事：对抗宏大世界，改变二战。     我觉得躲起来、躺平比较心安理得，懦弱一点、指责别人指责世界，不用殚精竭虑。          但这几个月以来，我时常回想起这封信，对我的改变很大，虽然这封信讨论的是二战，但换成我自己的工作和生活，这个概念是适用的。比如说，个人在公司的环境里，有多大的能动性、主动性，与其坐着抱怨环境，不如用自己的行动和影响力去改变环境，如果真的改变不了，到时再认。 |
| 《遥远的救世主》           （1）一开始在抖音上看，以为是故作高深的言情小说。后面发现越读越觉得有意思。在小说的框架内包含了佛教、哲学、市场经济、文化属性……每个模块都是一个大学问，需要钻研很久，值得多读几遍      （2）作者通过文中主人公口讲述"神即道，道法自然，如来"，给出了作者自身对于神/道/佛的理解——神是什么，是客观规律。客观规律遵从自然规则。不管是神、道、如来，最终都是让我们回到认识自己，承认自己的局限，打破世俗所认知的神、佛、道等种种依靠和妄想，从而悟出一份平常心，实事求是      （3）对于文化的理解——中华民族总是以有文化自居，但却忘了问一句，是有什么文化？是追求真理真相的文化还是弱势文化？是符合事物规律的文化还是违背事物规律的文化？任何一种命运，都是文化属性的产物，不以人的意志为转移 |
| 《混乱的猴子 :  硅谷的肮脏财富与随机失败》     作者的人生哲学是：因为只有物理的真理才是真理，所以其他事情上都可以自由发挥。      结合今年我生了孩子，体验过一段时间整个身体裂开后，觉得生命的宽度被打开了。孩子某种程度上延长了对未来的认知，作者的这个观点也让我对下限这个词有新的拓宽。现在比之前过得更踏实更充实，也更自洽了。 |
| Stoner（斯通纳） by  John Williams。我在audible上听的有声书版本由Robin Field朗读，古典的吐字带领我走过斯通纳如梦又似暗涌的一生。 |
| 2021年读过最好的书应该是《福格行为模型》，通过本书的福格模型-B=MAP（行动=动机&能力&提示），让我对于日常行为有了一个系统的认知，通过该框架能够让我如何化抱怨为行动，并且逐渐很顺畅的养成一些微习惯。 |
| 《系统之美》。一年前读不懂，今年再看突然有了明悟。          这是一本系统思考的指南书，如何理解身边的系统，如何分析身边的系统。小到玩具，大到全球性复杂挑战。          这本书不仅有基本概念，还有特征，陷阱，对策，变革方式和生存法则，是一本很好的工具书。          如果能够理解系统思考，有助于我们去分析问题，有最实在的，能够帮助我们做决策。          而我推荐这本书的原因真的很简单，因为我去年读不懂，今年开始有点懂了。          刚读书的时候，念的是法学，根本不理解什么是主体，关系，概念，原则、结构，效力，目标，那时候觉得自己懂了会考试，其实啥也不懂。          毕业五年，回过头来，终于才开始看懂了。          终于才理解一点点，什么是系统，才开始有了建构的意识。          可能是第三次意识觉醒。经验+这本书的引导，觉得自己幸运。 |
| 《超级转化率》一本很实用的工具书，用这本书做业务，利用持续改善思想，帮助客户在线上的数成功实现据增长。 |
